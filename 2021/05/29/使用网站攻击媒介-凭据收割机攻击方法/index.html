示例使用 Kali

# 使用 SEToolkit (setoolkit)

在命令行中输入 setoolkit

# 选择 Social-Engineerring Attacks 社会工程学攻击 (1)

选择 1

说明

1）社会工程学攻击
2）渗透测试 (快速跟踪)
3）第三方模块
4）更新社会工程学工具包
5）更新 SET 配置
6）帮助，积分和关于
99）退出

# 选择 Website Attack Vectors 网站攻击向量 (2)

选择 2

说明

1）鱼叉式网络钓鱼攻击向量
2）网站攻击向量
3）传染性媒体生成器
4）创建有效载荷监听器
5）群发邮件攻击
6）Arduino 攻击向量
7）无线接入点攻击向量
8）QRCode 生成器攻击向量
9）Powershell 攻击向量
10）第三方模块
99）返回主菜单

# 选择 Credential Harvester Attack Method 凭证收集器攻击方式 (3)

选择 3

说明

1）Java Applet 攻击方法
2）MetaSploit 浏览器漏洞利用方法
3）凭证收集器攻击方法
4）Tabnabbing 攻击方法
5）Web Jacking 攻击方法
6）多重攻击 Web 方法
7）HTA 攻击方法
99）返回主菜单

# 选择 Site Cloner 站点克隆 (2)

选择 2

说明

1）网页模版
2）站点克隆
3）自定义导入
99）返回 Webattack 菜单

# 输入回传到的地址 (默认即可)

# 输入要克隆的网站的网址

# 克隆好后会显示在本地打开的端口进行访问

访问 80 端口就是被克隆后到站点 (乱码先不管)

# 输入信息登录后会显示在控制台信息

在克隆到站点上输入用户名密码点击登陆

在控制台就会输出输入的用户名密码信息
